# 发布说明

软件包：wps-read-aloud-zhangjingyao
版本：1.0.15
架构：arm64
开发者：zhangjingyao
发布时间：20260516

## 本次变更

- 增加 Ribbon 控件启用状态：未朗读时“停止朗读”置灰，朗读中“开始朗读”置灰。
- “朗读方式”改为下拉框，包含“连页朗读”和“当页朗读”两个选项，默认连页朗读。
- 朗读过程中“朗读方式”和“朗读语速”置灰不可切换，避免播放状态和用户设置不一致。
- 朗读语速精简为 `0.75x`、`1x`、`1.5x` 三档。
- 修复连页/当页使用 toggleButton 在 WPS 2023 Linux 中触发“未知按钮”的兼容问题。
- 关于弹窗中的说明文件改为在同一弹窗内打开，可查看内容并返回关于页。
- 重新制作 Ribbon 单色大图标，减少透明边距，让图形主体占满图标区域。
- 开始朗读时显示“稍等片刻，朗读正在启动”提示，提示会自动关闭，进入播放后也会主动关闭。

## 安装

```bash
sudo dpkg -i wps-read-aloud-zhangjingyao_1.0.15_arm64.deb
```

如果 WPS 已经打开，请安装完成后重启 WPS。

## 已知限制

- 当前页范围识别依赖 WPS 2023 for Linux 对页码定位接口的支持。如果目标 WPS 页码接口不可用，当页朗读会尽量回退到当前光标到文档末尾的可读范围。
- 正在朗读时不能切换朗读方式和语速，需要停止后再调整。
- 音量由系统音量、声卡或桌面环境声音设置控制，加载项内不提供音量调节。
