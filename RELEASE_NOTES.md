# 发布说明

软件包：wps-read-aloud-zhangjingyao
版本：1.0.13
架构：arm64
开发者：zhangjingyao
发布时间：20260516

## 本次变更

- 删除“朗读音量”菜单和相关服务端音量缩放逻辑，朗读音量统一由银河麒麟系统音量控制。
- “朗读语速”保留为 Ribbon 下拉菜单，并增加当前选项的选中状态显示。
- 新增服务端朗读会话接口：`/read/start`、`/read/status`、`/read/settings`、`/read/pause`、`/read/resume`、`/read/stop`。
- 朗读时由本地服务维护播放状态，前端轮询当前句序号并同步选中 WPS 文档中的当前语句。
- 服务端默认预合成后续 3 句，降低句与句之间的等待间隔。
- 朗读过程中修改语速后，会清理未播放的预合成缓存，并从下一句开始按新语速重新合成。
- 继续使用 `vits-zh-hf-fanchen-C` 单模型方案，优先保证中文文档朗读清晰稳定；英文单词在预处理阶段按字母间隔方式送入中文模型。
- 发布日期和版本号统一更新为 `1.0.13 / 20260516`。

## 安装

```bash
sudo dpkg -i wps-read-aloud-zhangjingyao_1.0.13_arm64.deb
```

如果 WPS 已经打开，请安装完成后重启 WPS。

## 已知限制

- 当前句子选中和自动翻页依赖目标 WPS 版本对 `Range.Select()` 和 `ScrollIntoView()` 的支持情况。
- 当前语速修改不会改变正在播放中的同一句音频，会从下一句开始生效，以避免播放中途重采样导致破音或卡顿。
- 音量不再由加载项控制，请使用系统托盘、声卡或麒麟系统声音设置调整。
- `vits-zh-hf-fanchen-C` 更适合中文正文朗读；英文内容以清晰拼读为目标，不追求英文母语连读效果。
