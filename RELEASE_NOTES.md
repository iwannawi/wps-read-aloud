## 变更

- 新增 WPS 选区一致性自动化测试，固定使用《维护手册.docx》验证完整文档内容。
- 验收标准升级为完整文档测试，并补充“朗读文本、WPS 当前选区文本、当前显示页”三项一致性检查。
- 测试场景新增从其他页当前光标连页朗读、当页朗读、目录页、正文页、跨页长句和表格单元格。
- Windows 和 Linux 朗读预合成窗口统一调整为 240 字或 8 句，提升短句、目录和高速朗读场景下的连续播放能力。
- Windows 合成并发调整为最多 3 个 Sherpa-onnx 进程，减少下一句等待时间，同时保留只等待第一句即可开始朗读的启动策略。
- 弹窗关闭流程撤销主动抢焦点逻辑，不再调用窗口激活、前台切换或 Win32 焦点恢复接口，避免干扰 WPS 原生弹窗返回行为。
- 启动朗读提示改为创建朗读会话后再以 WPS 模态对话框显示，自动关闭路径与状态检查、关于朗读保持一致。
- 发布脚本和经验文档补充编码处理要求，避免 Windows PowerShell 5.1 误写 UTF-8 中文文件。

## 修复

- 修复 Windows 安装界面在部分 DPI、分辨率或字体环境下，安装路径、进度信息和日志输出框可能重叠的问题。安装界面现在使用更大的内容区域，并在小屏幕下提供滚动。
- 修复 Windows 安装界面宣传图显示不完整的问题。宣传图现在按比例完整显示，不再裁切上半部分。
- 修复前序版本为处理弹窗焦点而额外激活 WPS 的副作用。现在弹窗关闭后交回 WPS 原生窗口机制处理，不再主动最小化、恢复或抢占前台。
- 修复 WPS 目录域中直接调用 Range.Select 可能选中整段目录的问题。现在会在选中文本与朗读文本不一致时，自动使用 WPS Find 精确定位当前朗读条目。
- 修复 WPS 表格单元格中控制字符导致当前选区多选相邻单元格的问题。现在只在当前朗读片段附近精确查找，不再全文查找同名文本，避免表格中重复文字选错单元格。
- 修复朗读到空表格单元格时会朗读“空白内容”的问题。空单元格现在直接跳过。
- 修复文档内图片、嵌入对象等非文本元素可能进入朗读和选区范围的问题。非文本对象现在作为分隔符跳过，不朗读也不选中。
- 修复跨页长句测试误用句末页码判断当前页的问题。现在按句首所在页验证选区显示，更符合实际朗读行为。
- 修复 Windows 端极短目录文本重试时被截断错误信息误判的问题，确保“目录”等短文本失败后能继续尝试带上下文合成。

## 已知限制

- 当前句选中和翻页依赖 WPS Office 对文档 Range、Selection、Page 等接口的支持。
- 朗读过程中不能切换朗读方式和语速，需要停止朗读后再调整。
- 音量由目标操作系统、声卡设备或桌面环境声音设置控制，加载项内不提供音量调节。
